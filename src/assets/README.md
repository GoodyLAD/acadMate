# Assets Directory

This directory contains static assets for the application:

- `images/` - Image files (PNG, JPG, SVG, etc.)
- `icons/` - Icon files
- `fonts/` - Font files
- `videos/` - Video files
- `documents/` - Document templates and samples

## Usage

Import assets using the `@/assets/` alias:

```typescript
import logo from '@/assets/images/logo.png';
import icon from '@/assets/icons/user.svg';
```
